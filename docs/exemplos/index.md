# 💡 Exemplos de Uso

Exemplos práticos de como usar o `suap-api-wrapper` como biblioteca Python em situações reais.

---

| Exemplo | Descrição |
|---|---|
| [📊 Relatório de Notas](relatorio-notas.md) | Gera um resumo das notas e frequência do semestre atual |
| [🤖 Bot de Notificação](bot-notificacao.md) | Envia alertas de notas via Telegram |
| [📁 Exportar para CSV](exportar-csv.md) | Exporta diários e notas para um arquivo `.csv` |

!!! tip "Pré-requisito"
    Todos os exemplos assumem que você já executou `suap login` e possui uma sessão salva em `~/.suap/`.
